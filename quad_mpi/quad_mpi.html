QUAD_MPI
Parallel Computation of Integral Estimate

QUAD_MPI is a PYTHON script which estimates an integral with multiple MPI processes, using MPI4PY for parallel execution.

Languages:

QUAD_MPI is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a Python version.

Related Data and Programs:

HELLO_MPI, a PYTHON script which prints out "Hello, world!" using the MPI parallel programming environment, under MPI4PY.

PRIME_MPI, a PYTHON script which counts the number of primes between 1 and N, using MPI for parallel execution.

QUAD_SERIAL, a PYTHON script which approximates an integral using a quadrature rule, and is intended as a starting point for parallelization exercises.

SEARCH_MPI, a PYTHON script which searches integers between A and B for a value J such that F(J) = C, using MPI and MPI4PY for parallel execution.
